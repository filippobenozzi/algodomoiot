Sheltr ESP32 Web Flasher

Apri questa pagina con Chrome/Edge desktop, collega ESP32 via USB e avvia il flash.

  1. Esegui prima lo script build sul server: ./build_esp32_firmware.sh
  2. Collega ESP32 via USB
  3. Premi il pulsante qui sotto e seleziona la porta seriale
